# Field Notes / Lore Codex Risks

## Runtime Risks

- Long text overflow.
- Menu state bugs.
- Text speed / window conflict.
- Localization / string storage bloat.
- Scope creep into PokeNav.
- Save unlock complexity if added too early.

## Risk Controls

- Keep the MVP text-only.
- Use short entries.
- Avoid unlock tracking.
- Avoid new data formats until the viewer behavior is validated.
