# Field Notes / Lore Codex

## Status

- Status: Planned
- Code status: No code changes on master
- Docs status: Initial design
- Feature type: Fully new local feature candidate

## Goal

Create a small in-game archive for worldbuilding notes, developer notes,
ecology notes, local myths, or region records.

## First MVP

- Open from NPC or safe test object.
- Show 3 fixed entries.
- No unlock state.
- No save data.
- No PokeNav integration in first slice.
- Text-only viewer is acceptable.
- Entries can be static strings.

Example entry categories:

- Hoenn Myths
- Local Ecology
- Trainer Culture
- Ancient Ruins
- Battle Frontier Notes
- Developer Notes

## Non-goals

- No unlock tracking in MVP.
- No quest dependency.
- No new save fields.
- No large UI framework.
- No PokeNav page in first slice.
