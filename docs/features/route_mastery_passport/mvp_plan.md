# Route Mastery Passport MVP Plan

## First Slice

- Route 101 / Route 102 or another safe candidate.
- Small fixed route table.
- Checklist UI.
- No dynamic discovery.
- No completion rewards unless separately approved.
- No save layout changes.

## Checklist Contract

| Item | MVP source |
|---|---|
| Visited | TBD; omit if no safe existing flag exists. |

| Hidden items found | Fixed route-specific hidden item flag list, if safe. |
