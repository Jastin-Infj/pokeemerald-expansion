# Route Mastery Passport Risks

## Runtime Risks

- Flag ownership confusion.
- Trainer rematch flags.
- Hidden item behavior.
- Item ball flag naming.
- Route sections with multiple maps.
- Save layout temptation.
- Too much manual data entry.

## Risk Controls

- Start with one route only.
- Use a fixed route data table.
- Avoid rewards in the first slice.
- Do not infer completion from broad flags without route-specific audit.
