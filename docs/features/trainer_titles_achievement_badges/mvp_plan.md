# Trainer Titles / Achievement Badges MVP Plan

## First Slice

- NPC title clerk.
- 3 fixed titles.
- Event flag based ownership.
- No active equipped title in MVP.
- No Trainer Card changes.
- No save layout changes.

## MVP Title Flow

| Step | Behavior |
|---|---|
| Check | NPC checks existing or newly allocated event flags. |
| Grant | NPC grants a title flag if a simple condition is met. |
| Display | NPC lists earned titles as text. |
| Repeat | Already earned titles do not grant twice. |

## Future Work

- Active title save state if safe.

- Badge page / achievement grid.
