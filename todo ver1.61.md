# todo v1.61（タグ抽選）

## 目的
- トレーナーのスロットにタグ条件を付け、戦略性の高いパーティ構成を作れるようにする。
- タグ一致のみを抽選し、**フィルタ後の重み合計**で抽選する（重み無視はしない）。

## 決定事項
- **タグ定義は pool 側**（`data/trainer_rank_party/*.party` の `Tags:`）。
  trainers.party 側は **OR/AND 必須**でフィルタ指定する（省略NG）。
- **タグはスロット内にのみ記述**。トレーナーヘッダーは **ビルドエラー**。
- **1スロットにつきタグ系は1種類のみ**（`Tags` / `LeadTags` / `AceTags` の併用はNG）。
- **タグ付きの非ランク/非専用スロットはビルドエラー**。
- タグ一致0件は **ビルドエラー優先**。万一ランタイムに残った場合は **バニラ固定**。
- `LeadTags/AceTags` は抽選フィルタとして使い、**順序は自動調整**（Lead 先頭 / Ace 最後）。
